    Chapter 9: The Window Object

  • Every JavaScript environment has a global object.
  • The Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen.
  • You can use the BOM to see what pages were viewed before they the came to yours.
  • Don’t rely on this information though, as it can be modified by a user to masquerade as a different browser. We should use a different method of determining what browser the user is using.
  • window.location.href will return the URL that is in te browser search bar.
  • The reload() method can be used to force a reload of the current page. If it’s given a parameter of true, it will force the browser to reload the page from the server, instead of using a cached page.
  • The toString() method returns a string containing the whole URL.
  • Using window.history.length will allow you to see how many pages were looked at before arriving at the current page.
  • We can also hard code a back button so that the the user will move back or a forward button so that the user will move forward in the browser.
  • Using window.open(), we can open up a new window and provide a width and height as well as an exact URL where we want to take the user.
  • The close() method can be used to close a window, assuming you have a reference to it.
  • The Screen object has more uses for mobile devices. It also allows you to do things like turn off the device’s screen, detect a change in its orientation or lock it in a specific orientation.
  • You actually have to assign a cookie to the JavaScript Cookie Jar.
  • To remove a cookie, you need to set it to expire at a time in the past.
  • The window.setInterval() method can be used to repeatedly invoke a function. It can be stopped using the clearInterval() method.

  • The position object has several other properties that can be used to find out information about the location and movement of the device: position.speed property returns the ground speed of the device in meters per second. position.altitude property returns an estimate of the device’s altitude in meters above the WGS84 ellipsoid, which is a standard measurement for the center of the Earth. position.heading property returns the direction the device is moving in. This is measured as a bearing in degrees, clockwise from North. position.timestamp property returns the time that the position information was recorded.
