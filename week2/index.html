    Chapter 2: Programming Basics

  • The reason that var wasn't totally removed from the language was because JavaScript needs to remain backwards compatible.
  • When using the dot notations with numbers it is a good practice to use two dots so that the computer does not confuse the dot with a decimal point. There are many other ways to avoid this, but I like the double dot the most.
  • It matters where you out the '++' or '--' because if you do ++points, the program will first increase points and then return it. Points++ will return points and then increase the value.
  • Placing a '+' in front of "5" will turn it into a number.

    Chapter 3: Arrays, Logic, Loops

  • You can look at the values of an array in the console by simply typing the name of the array in the console.
  • Using the join method with an array will turn all of the elements in the array into a long string.
  • The slice method allows you to start and end with a specific index of an array.
  • You can find if an array contains a value using the indexOf operator.
