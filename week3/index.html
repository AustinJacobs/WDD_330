    Chapter 6: The Document Object Model

  • The Document Object Model, or DOM for short, represents an HTML document as a network of connected nodes that form a tree-like structure.
  • This means it can be used in any programming language, not just JavaScript. I did not know this. I always thought that the DOM was only accessed with JavaScript and that there was no other way.
  • It is super cool that you can access all of the elements on a page with the DOM and perform an action. This code could be very helpful: for (let i=0 ; i < document.images.length ; i++) { // do something with each image using document.images[i] }
  • You can get and set attributes of an HTML element with the DOM.
  • Code to create a function that makes elements on the HTML page: function createElement (tag,text) { const el = document.createElement(tag); el.textContent = text; return el }
  • Every element node has a style property. This can be used to dynamically modify the presentation of any element on a web page.
  • Camel case must be used when accessing css properties that normally contain a hyphen.
